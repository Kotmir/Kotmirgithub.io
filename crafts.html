Уникальные крафты

Специальные рецепты, доступные только на сервере Lumix. Уникальные крафты добавляют разнообразия в игровой процесс и открывают новые возможности для творчества и строительства.

Декоративные

Блок света

Источник света, который не виден как блок, но освещает область вокруг себя. Идеально для скрытого освещения в builds.

Ингредиенты:
8х стекло 1х светокамень
Инструменты

Крылья Lumix

Уникальные крылья с улучшенной маневренностью и специальными визуальными эффектами. Позволяют парить в воздухе дольше обычного.

Ингредиенты:
2х сердце моря 2х кристалл энда 2х скалк 1х элитры 1х скалковый крикун 1х голова дракона
Декоративные

Светящаяся рамка

Рамка для предметов, которая светится в темноте. Отлично подходит для выделения важных предметов в ваших builds.

Ингредиенты:
Рамка ×1 Светокамень ×4 Золотой слиток ×4
Декоративные

Невидимая рамка

Рамка, которая становится невидимой после размещения, оставляя только предмет внутри. Создает эффект парящего предмета.

Ингредиенты:
Рамка ×1 Стекло ×4 Эндер-жемчуг ×2
Функциональные

Звезда Пустоты

Мощный компонент для создания уникальных предметов. Используется в различных продвинутых крафтах на сервере.

Ингредиенты:
2х незеритовый блок 2х череп визер скелета 2х плачущий обсидиан 1х осколок эха 1х звезда незера 1х пластинка 11
Строительные

Укрепленный глубинный сланец

Прочный строительный блок с высокой взрывоустойчивостью. Идеален для создания защищенных конструкций и подземных баз.

Ингредиенты:
Глубинный сланец ×4 1х скалк 1х скалковый блок 1х скалк сенсор 1х скалковый крикун
Функциональные

Паутина

Декоративный блок, который также замедляет движение. Полезен для создания ловушек и декоративных элементов в builds.

Ингредиенты:
Нить ×8 1х слизь
Функциональные

Черный краситель

Альтернативный способ получения черного красителя. Полезен когда чернильные мешочки сложно найти.

Ингредиенты:
Уголь ×1
Функциональные

Нитки

Альтернативный способ создания ниток. Полезен когда пауки недоступны или для массового производства.

Ингредиенты:
Белая шерсть ×1
Строительные

Красный песчаник

Декоративный строительный блок из красного песка. Альтернатива обычному песчанику для разнообразия в строительстве.

Ингредиенты:
2х песчаник 1х красный краситель
Функциональные

Красный песок

Альтернативный способ получения красного песка. Полезен когда месса недоступна или для экономии времени.

Ингредиенты:
Песок ×8 Красный краситель ×1
Функциональные

Гравий

Альтернативный способ получения гравия. Полезен когда природные залежи гравия недоступны или исчерпаны.

Ингредиенты:
4х кремень
